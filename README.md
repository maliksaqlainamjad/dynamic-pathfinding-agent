# Dynamic Pathfinding Agent

An interactive grid-based visualizer that compares A* and Greedy Best-First Search (GBFS) algorithms in real time.

## Features
- Switch between A* and GBFS algorithms
- Choose heuristic: Manhattan or Euclidean distance
- Place/remove walls manually or generate a random map
- Dynamic mode: obstacles spawn mid-run, forcing the agent to re-plan
- Live metrics: nodes visited, path cost, time (ms), re-plans

## How to Run
Make sure Python is installed, then run:
python pathfinding_agent.py

## Built With
Python, Tkinter (no external libraries required)
